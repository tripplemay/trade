## 2. ★★本批次最重要的设计决定：从第一天就拆腿

B103-B105 的教训是本仓最贵的一条：`inst_buy_net` 的横截面 IC ~0.15 看起来很好，
**但纸面收益的 90–107% 落在 A 股不可做空的短腿上**，可落地的 long-only 绝对 edge ≈ 0。
那次是 Codex 在验收阶段拆腿才发现的——**如果只看 IC，会得出完全相反的结论**。

因此本批次冻结以下口径，不得在实施阶段协商：

| | 规定 |
|---|---|
| **主判据** | **long-only 顶层组合相对全宇宙等权基准的超额**。这是唯一能落地的形态（¥210 万现金纯多头账户，无融券） |
| **IC 的地位** | **辅助诊断，不作判据**。IC 高而多头腿无超额 = NO-GO，不是 INCONCLUSIVE |
| **强制拆腿** | 多空组合收益必须拆成「多头腿相对基准」+「空头腿相对基准」两项分别披露；空头腿贡献占比须显式给出 |
| **单调性** | 必须查 E/P 分组的**单调性**，不得据 top-bottom 同号就称方向真实（B077 驼峰教训：极端组均值回归） |

## 3. 冻结的计算口径

### 3.1 分子：PIT TTM（按 handoff §6 唯一算法）

```text
SQ1 = C_Q1
SQ2 = C_H1 - C_Q1
SQ3 = C_Q3 - C_H1
SQ4 = C_FY - C_Q3
TTM = 最近连续四个 SQ 之和
```

- **四个分量必须各自独立 as-of**（同一 `formation_date` 重新解析），不得混用不同知识截止日。
- 等价式交叉验证：`FY(y-1) + YTD(y) - YTD(y-1)`。两式不一致 → 失败码，不取其一。
- 财年变更 / IPO 历史不足 / 期间不可比 → 返回 null + 原因码。
  **禁止**用最新年报、线性插值、季度年化填补。
- 全程 Decimal，导出前才转浮点。

★B109 F001 实测：**修订风险集中在 FY 分量**（FY 0.747% vs Q1 0.073%，10.2 倍）。
`ep_feature_component` 必须能区分四个分量各自的版本与风险，不得用混合平均掩盖。

### 3.2 分母：B109 已验证的 `CN_SECURITY_TOTAL_MV`

直接复用 `scripts/research/ashare_pit/marketcap.py`。G4 已由 F003 独立验收
（27,963 证券-日 100.0000% 通过身份校验）。**禁止**流通市值（禁令 #6）。

### 3.3 宇宙与形成日

- 复用 `universe.py`：全状态 L/D/P + 含退市名（禁令 #11，B109 实测幸存者偏差 8.91%）。
- 形成日 = 2013-01 至 2024-12 **每个自然月末**，连续 144 个月，不得向后滑动起点。
- E/P = TTM / total_mv。

### 3.4 ★负 TTM 的双口径（用户 2026-07-20 冻结）

同时跑两个口径，**主口径先冻结，禁止事后挑好看的那个**：

| 口径 | 定义 | 地位 |
|---|---|---|
| **主口径** | **不剔除**负 TTM，单独成组披露其收益 | **§4 裁定以此为准** |
| 对照组 | 剔除负 TTM 后重新分组 | 仅作稳健性对照，**不得用于裁定** |

理由：剔除亏损股是**事后选择**——A 股亏损股常带壳价值与重组预期，剔除会系统性改变结论。
两个口径必须**并排呈现**，且报告须明确指出二者差异有多大（差异本身就是信息：
若剔除后结果显著变好，说明信号有相当部分来自「避开亏损股」而非「买便宜」）。

## 4. ★预注册裁定判据（须在看到任何结果前冻结）

预注册的意义在于**锁死之后不能改**。看到结果再定标准，等于没有标准。

**★用户已于 2026-07-20 批次启动前冻结如下，实施阶段不得协商：**

| 档 | 条件（2013-2024 全区间） |
|---|---|
| **GO** | long-only top-quintile 等权、月度调仓，相对全宇宙等权基准**年化超额 ≥ 3.0%**，且**分年正超额年份 ≥ 60%**，且 E/P 五分组**单调** |
| **NO-GO** | 年化超额 **≤ 1.0%**，或分组显著非单调（顶层组不是最优） |
| **INCONCLUSIVE_COVERAGE_LIMITED** | 落在 1.0%–3.0% 之间且方向一致；**或**联合覆盖率 < 50%（framework `generator.md §37`） |

**为什么是 3.0% 而不是 2.0%：** 本批次**不含交易成本**，而 A 股月度调仓一个五分位组合的
成本约 **1–2%/年**。若 GO 线定在 2%，通过时净超额可能只剩 0–1%——等于用一个几乎不赚钱的
结果批准了后续大额工程投入。**3% 毛超额才对应约 1–2% 净。**

> 预注册只有锁死才有意义。看到结果后调整任何一档，本批次结论作废。

## 5. ★诚实限制（必须随任何结果一同呈现）

本批次**不做**以下事项，因此**任何正向结果都是上界，不是可实现收益**：

- ❌ 无行业中性 / 规模中性（handoff §4.6 的 PIT 行业根本不存在）
- ❌ 无停牌 / 涨跌停 / ST 可执行性检查（handoff G3b 未建）→ 有些"买入"实际买不到
- ❌ 无交易成本、无冲击成本、无换手率约束
- ❌ 无公司行动处置（handoff §4.7 未建）→ 退市证券的最终现金去向未建模
- ❌ 不是 handoff 的 G5/G9 级 lineage 与连续面板

**结论只能用于「值不值得继续建地基」这一个决策，不得外推为收益预期。**
