## 4. ★关于「等价式交叉验证」的诚实结论（必须随任何结果一同转述）

spec §3.1 要求同时算两条路径并比对：路径 A（四单季差分之和）与路径 B（`FY(y-1)+YTD(y)−YTD(y-1)`）。

**结论：在 Decimal 精确算术下二者严格恒等，鉴别力为零。**

```
路径A = Σ_{j=k+1..4} SQ(y-1,j) + Σ_{j=1..k} SQ(y,j)
      = [C_FY(y-1) − C(y-1,k)] + C(y,k)
      = C_FY(y-1) + C(y,k) − C(y-1,k)          ← 逐字就是路径 B
```

中间累计项全部望远镜相消。已由代数展开 + 单测（多组任意 Decimal × 四个锚点穷举）双重确认。

**三条推论：**
1. 一致率 100% **不构成任何数据质量证据**。★**禁止**在任何报告中写「数据已交叉验证」。
2. 它连「某分量有 10 倍单位错」都抓不到——中间项在路径 A 内部自我抵消。
3. 保留该检查是因为 spec §3.1 冻结了它、且成本为零；它唯一能触发的是「路径 A 缺分量而
   路径 B 不缺」，那是**覆盖率事件**，必须单独计数，不得贴「数据完整性」标签。

> 这是与 B109 审计器「可裁定样本仅 16.7% 而一致率 100%」**同型**的自欺陷阱：
> 分母被自己缩掉之后，一致率不再有信息量。

**替代的真校验（F001/F002 实做）：**

| 代号 | 检查 | 判据 |
|---|---|---|
| **R1** | **TTM 阶跃性**（唯一直接检验 PIT 性）：`TTM(i,t) ≠ TTM(i,t−1M)` 时，必存在某分量满足 `t−1M < f_ann_date ≤ t` | 违反 → `TTM_STEP_WITHOUT_FILING`。抓 as-of 漏进未来 vintage / 锚点抖动 / 缓存串期 |
| **R2** | **`report_type=2` 单季直报对拍**（D4） | `abs(SQ_diff − SQ_direct) > abs(SQ_direct)×0.001 + 1000` → `CUMULATIVE_BASIS_BREAK` |
| **R3** | `daily_basic.pe_ttm` 反推的准外部 anchor | ★只在 **latest-vintage** 下比。**致命限制：`pe_ttm` 亏损为空 → 负 TTM 组永远没有外部 anchor**，而 §3.4 主口径恰恰不剔除负 TTM = 校验覆盖被被校验变量的符号所选择 |
| **R4** | 量级闸门 `0.005 ≤ median(E/P) ≤ 0.20` | 抓万元/元单位错。注意单位错**不改变横截面排序**，主判据一分不差 |

## 5. ★★铁律：任何以「数值本身」为条件的置空一律禁止

实测：2021 年四期齐全的 4,716 只证券里，**1,786 只（37.87%）至少有一个负单季**，
1,272 只（26.97%）SQ4 为负，731 只（15.50%）TTM 为负（2015 年同口径 38.90% / 21.55% / 11.91%）。

把「负单季 / 累计值倒退」当异常置空，会一次性抹掉约 38% 的宇宙，且抹掉的正好是亏损与
减值集中的那一端。spec §3.4 要求负 TTM 不得剔除，但**杀伤点在上游一级**：负 SQ 若被当作
数据错误拦掉，负 TTM 根本不会诞生，「双口径」设计在实现层即被架空。

> **A 类（可 fail-closed）只能来自元数据**：日期、报表类型、期间连续性、宇宙成分、币种。
> **B 类（有效经济事实，必须保留进组）**：一切关于数值方向、大小、同比跳变的判断。
> **★code review 硬检查项：搜代码里有没有以符号 / 量级为条件的置空。**
> 这是本设计里唯一**会系统性改变结论、却完全不体现在覆盖率上**的错误。

附证：`|SQ| > 10×|C_FY|` 在 2021 有 22 例，绝大多数由 `C_FY ≈ 0` 造成
（`605333.SH` FY = −1,056,223 而 SQ4 = −46,408,573）。**任何以 FY 为标度的量级检验都是坏的。**
