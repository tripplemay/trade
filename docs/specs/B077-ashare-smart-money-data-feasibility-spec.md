# B077 — A股 聪明钱数据可行性摸底（机构/大资金流 → 能不能支撑跟随策略）Spec

**批次定位：** 新方向第一步——研究/可行性摸底。用户真实目标=**跟踪机构/大资金,摸清其建仓动向,跟随获利的实战策略**（不同于 cn_attack 价格+财务因子，是"信息流/聪明钱"族）。本批**不搭策略**,先摸地基:VM 实测 A股 聪明钱数据**现在到底能拿到什么、够不够用**+ 初步看**有没有 edge 苗头** → 出"可用信号清单 + 数据现实 + 推荐先搭哪个"的诚实结论。

**来源：** 2026-06-25 用户澄清真实目标（跟踪机构/大资金、摸其策略、跟随建仓盈利）→ 选「先做可行性摸底再定」。

## 0. ★诚实约束（焊死）

- **§23 数据现实必须 VM 实测,不得假设**：A股 聪明钱数据近年披露口径有变——**北向资金 2024.8 取消盘中实时披露**（改收盘/延迟/持股级）→ 经典"盘中跟北向"已弱。**到底现在 akshare 能拿到什么字段/时效/历史深度,只能 VM 实跑验证**（北向/龙虎榜/资金流各一）。
- **first-look ≠ 可交易 edge**：F002 的信号探查是**初步相关性苗头**（IC / 分组 forward-return spread），**不是全回测、不是 verdict**。有苗头 → 下批搭策略全回测;无苗头 → 诚实劝退。**不承诺赚钱。**
- **去偏**：信号探查用 B070 去偏 PIT 宇宙（聪明钱常涌入小盘,小盘幸存者偏差最重）。
- **信号稀疏/覆盖差异**：龙虎榜仅覆盖异动股（稀疏）;资金流覆盖全市场（日级）;北向仅 Connect 标的（偏大盘）。F001 须如实记每源覆盖/稀疏度。
- **纯研究边界**：research-safe / no-broker / no 真金 / no 自动下单 / 只读公开披露数据（§12.10.2）;本批无生产改动、无策略部署（搭策略是下批,若 GO）。"实战/配资"前提=未来扛过同等去偏检验,本批不触及。

## 2. Feature 拆解（3 features：2 generator + 1 codex）

### F001 — 聪明钱数据可得性 VM 实测（§23，三源数据现实报告）（executor: generator）

1. **建探针 loader**（照 cn_*.py 范式，lazy akshare、best-effort）：北向持股 / 龙虎榜机构席位 / 主力资金流，各一。
2. **VM §23 实跑**：每源出**数据现实**——字段（有没有"机构净买入/超大单净流入/北向持股变化"这种可用列）、时效（日级?延迟?）、**历史深度**（能回溯几年=能不能回测）、覆盖/稀疏度（全市场?仅异动?仅 Connect?）、稳定性（成功率/限流）。
3. **★北向 2024 披露变更现实**：实测现在 akshare 北向到底给延迟/持股级什么,经典实时玩法弱化程度。
4. 单测（探针解析/空帧守门，确定性 fixture）。

**Acceptance：** 三源探针 + VM 实测数据现实报告（字段/时效/历史深度/覆盖/稳定性 per 源 + 北向2024现实）;明确每源"能不能支撑回测"。Gates：backend pytest/ruff 目录上下文/mypy CI-exact 0。

### F002 — 初步信号质量探查（first-look，去偏宇宙，推荐先搭哪个）（executor: generator）

1. **信号构造**（对 F001 判定可用的源）：把聪明钱数据转成 per-stock 横截面信号（机构净买入额/分位、超大单净流入、北向持股增幅）。
2. **first-look IC / 分组 forward-return**（B070 去偏宇宙）：信号当日 vs 未来 N 日收益的 IC / 分组 spread——**只看"有没有相关性苗头",非全回测**。处理稀疏（龙虎榜仅异动日）。
3. **推荐**：哪个源信号苗头最强、数据最扎实 → 下批搭策略的首选;或都没苗头 → 诚实劝退。
4. 诚实 caveat（first-look ≠ 可交易、信号拥挤/滞后、去偏宇宙残余偏差）。

**Acceptance：** 可用源的 first-look IC/分组 spread 真数字（去偏宇宙）+ 推荐先搭哪个（或都没苗头劝退）+ 诚实 caveat。Gates 同 F001。

### F003 — Codex 验收（独立复核数据可得性 + 信号探查真数字 + 推荐诚实）+ signoff（executor: codex）

**研究/可行性批次——signoff 含实测证据（§29 + §23）：**
- L1 全门禁（verifying 可跳 L1 复跑）。
- **VM 独立复核**：抽样复跑三源数据可得性（确认 F001 数据现实非夸大,尤其北向 2024 现实）;独立复跑信号 IC/spread（确认 F002 苗头真数字）。
- **★诚实推荐裁定**：哪个源 GO（值得下批搭策略）/ 全 NO-GO（都没苗头/数据不够）——不橡皮戳 generator。
- 边界：research-only/no-broker/no 真金/no 自动下单/只读披露数据;无生产改动。signoff 实测证据逐条 + 明确推荐 + caveat。
