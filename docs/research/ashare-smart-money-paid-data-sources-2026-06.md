> 独立调研任务（用户 2026-06-25 指派）。目的：B077 用免费源（akshare/baostock）发现三个缺口——北向已冻结、主力资金流历史太浅、龙虎榜机构席位小盘覆盖不足——调研**付费源能否补齐**。

## 结论（TL;DR）

1. **能覆盖,而且很便宜。** 标杆 = **Tushare Pro，约 ¥200–500/年**（积分制），直接补上 B077 两个可补缺口:龙虎榜机构明细（**2005 至今、全市场含小盘/退市**）+ 主力资金流（历史比 akshare 深）+ 全市场历史价（含退市,补小盘覆盖）。
2. **唯一救不回的是日度北向。** 这是**监管改的,不是数据获取问题**:2024-05-13 停实时、2024-08-20 停日度→改季度披露。**Wind 这种顶级源也只剩季度持股**。北向日度信号全市场都死了,别为它花钱。
3. **付费数据补的是"覆盖+深度",不是"有没有 edge"。** 它让 B078 那个决定性检验能在**全覆盖去偏样本**上真正跑出来,但**有没有超额收益,仍要去偏全回测说话**。别把"更好的数据"当"有 alpha"。

## 标杆推荐：Tushare Pro（为什么是它）

- **正好补 B077 两个可补缺口,且便宜到可忽略（¥200–500/年）。**
  - `top_inst`（龙虎榜机构交易明细）**2005 至今、全市场**——含 B070 去偏宇宙覆盖不到的小盘 + 退市名 → **直接解掉 80.8% 未覆盖**。
  - `moneyflow`（个股资金流：超大单/大单/中单/小单净流入）历史比 akshare 深 → 解掉"0.5 年太浅"。
  - 全市场历史日线（含退市）→ 重建**含小盘的去偏宇宙**,让 first-look/全回测 apples-to-apples。
  - `沪深港股通持股明细`（季度级北向持股,日度已死的唯一残值）。
- **接入成本低**：Python SDK,和现有 akshare lazy-loader 范式并列即可（多一个 paid loader,key 走 env）。

## 诚实 caveat（焊死）

1. **Tushare 是社区/捐赠（积分）模式,数据采自公开交易所**——质量足够做研究/回测,但**不是 Wind 级机构 SLA**。要institutional 级稳定性/审计才上 Wind（贵 100 倍）。
2. **付费数据不改边界**：仍 research-state / no-broker / no 真金 / no 自动下单;买数据 ≠ 可配资。
3. **付费数据 ≠ 有 alpha**：它解掉"测不准（覆盖/深度）",但"跟着机构到底赚不赚",仍要**去偏全回测**给真数字。付费只是让这个检验能正确地跑。
4. **北向别投钱**：日度信号监管性死亡,任何源都救不回。
5. **成本是年付**：Tushare 积分一年过期;Wind/聚宽是年费订阅。
