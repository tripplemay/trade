# A 股红利低波纯 E/P 数据闸门审计（2026-07-12）

> 数据裁定：**DATA_NO_GO**
> 是否计算候选收益：**否**
> 是否允许组合回测：**否**
> 资金口径：人民币 **2,100,000 元**，收益优先

## 1. 本轮研究方向

本轮不再检验 H30270 的 BP/EP/CFP 多价值替代，而是冻结一个新的、可归因的纯 E/P
候选：在 H30269 已完成连续分红、支付率、DPS 增长、高股息和低波筛选之后，只检验
额外的盈利收益率是否提高未来复利收益。

冻结定义如下：

- 母体：每次官方年度调整后实际生效的 H30269 历史成分及官方权重；禁止用当前成分
  回填，也禁止用 B070 的宽基宇宙替代。
- 信号：`最近四个已公开单季归母净利润之和 / 信号日总市值`。
- PIT：公告必须严格早于信号；累计利润先还原单季，未来修订不得覆盖历史版本。
- 构造：在 PIT 中证一级行业内按总市值相邻配对，将低 E/P 股票基准权重的 50% 转给
  高 E/P 股票，同时保持行业总权重、规模暴露和 15% 单股上限。
- 主窗口：`2013-12-31..2026-06-30`；发布前历史只作 backcast。
- 执行：下一可交易日开盘、100 股整手、无杠杆，处理停牌、涨跌停、退市、现金分红、
  公司行动和历史费用；单笔不超过信号日前 ADV60 的 1%。
- 数据通过后才启用收益门：净 CAGR 增量至少 2pp、Sharpe 增量至少 0.15、MaxDD
  恶化不超过 3pp、4 fold 至少 3 个为正、HAC `t>=1.65`、区块 bootstrap 下界大于 0。

理论依据仍来自 A 股纯 E/P 的研究，但理论先验不能替代数据门：
[Liu、Stambaugh、Yuan](https://doi.org/10.1016/j.jfineco.2019.03.008)
发现盈利价格比在多种 A 股价值代理中较稳健，同时明确要求使用形成日之前已经公开的
财报，并处理小市值壳价值污染。

## 3. H30269 历史成分与权重

仓库只有 H30269/H20269 指数点位，没有历史成分或权重文件。B070 则是沪深 300、
中证 500、上证 50 的季度并集，不能冒充 H30269。

中证官网免费材料比“只有当前十大”更完整：

- [当前完整样本](https://oss-ch.csindex.com.cn/static/html/csindex/public/uploads/file/autofile/cons/H30269cons.xls)：
  50 只，截至 `2026-07-10`。
- [最近月末完整权重](https://oss-ch.csindex.com.cn/static/html/csindex/public/uploads/file/autofile/closeweight/H30269closeweight.xls)：
  50 只，截至 `2026-06-30`，权重合计 99.998%。
- [当前十大权重接口](https://www.csindex.com.cn/csindex-home/index/weight/top10new/H30269)：
  当前更新日 `2026-07-10`。

但这些固定 URL 会滚动覆盖。向十大接口传入 `tradeDate=20141231`，返回的仍是
`2026-07-10` 当前数据，规范化哈希与不带日期的响应完全相同；路径追加历史日期则
404。它们不能构成 2013 年以来的 PIT 档案。

[中证指数数据服务](https://www.csindex.com.cn/#/dataService/indexData)将样本权重、公司
行动和指数除数列为静态数据；[授权资讯商页面](https://www.csindex.com.cn/#/dataService/vendor)
列有月度收盘权重和指数日静态数据。因此严格回测所需的历史包应通过中证数据服务或
授权商取得，不能抓取今天的文件后回填历史。

## 4. B068 `earnings_yield` 不是纯 TTM E/P

B068 基本面有 13,267 行、393 个当前幸存标的，但存在四个结构性问题：

1. `report_date` 只出现 `04-30 / 08-31 / 10-31`，是法定最晚披露日，不是真实公告日。
2. 3,301 组年报和一季报共用同一个 `04-30`；共 6,602 行 ticker-date 重复。
3. 生产逻辑使用累计基本 EPS 除以截止日附近价格，不是四个 as-filed 单季归母净利润
   除以总市值。
4. 没有真实公告日、版本号、归母净利润、总市值和 PIT 行业字段。

累计口径可从数据中直接看出。`earnings_yield / (1/PE_TTM)` 的季度中位比例为：

| 季度 | 中位比例 |
|---|---:|
| Q1 | 0.231 |
| Q2 | 0.515 |
| Q3 | 0.788 |
| Q4 | 0.969 |

比例随累计季度推进接近 1，证明它不是恒定的 TTM E/P；13,107 个可比记录中还有
752 个符号与 `1/PE_TTM` 不一致。

作为代理诊断，把 B068 基本面拼到 B070 去偏宇宙后：

- 1,310 个历史标的只有 **296 个**有基本面，union 覆盖 **22.60%**。
- 各期覆盖只有 **20.75% 至 31.75%**，远低于 95% 门。
- 536 个非当前成员只有 **45 个**有覆盖，会把退市和被剔除公司再次系统性丢掉。

用该交集运行 E/P 会重新引入严重存续偏差，因此本轮拒绝生成收益数字。
这组 20.75%–31.75% 是 **B068×B070 代理覆盖**，不是真实 H30269 覆盖；后者因
历史 H30269 母体缺失而不可计算，数据门直接 FAIL。

## 5. 实际公告面板：覆盖好，但版本错误

已有 `raw_reports.csv.gz` 是可复用的重要基础：

- 219,633 行、5,542 个原始证券代码，报告期从 2014Q1 至 2025Q4。
- `PARENT_NETPROFIT` 缺失仅约 0.006%。
- 对 B070 的 28 个已覆盖季度，报告覆盖 **94.13% 至 99.88%**；2026Q1 尚未进入缓存，
  与历史覆盖率分开记录。
- `NOTICE_DATE` 与独立实际披露表抽样 **12/12 一致**。

但这个免费接口每个 ticker-period 只保留一条当前历史值，没有原始版本序列：

- `UPDATE_DATE - NOTICE_DATE` 中位数 **365 天**，P90 **377 天**。
- **90.83%** 的记录在公告 120 天后仍更新，**41.12%** 超过 365 天。

所以日期是 PIT 的，数值却可能是后来修订的。把今天的归母净利润放回首次公告日，会
制造 revision lookahead。

[巨潮资讯公告检索](https://www.cninfo.com.cn/new/commonUrl/pageOfSearch?url=disclosure/list/search)
可以保留公告 ID、时间和原始 PDF，修订公告也作为独立记录存在；但公开网页接口没有经
验证的免费批量结构化 as-filed TTM 字段和 SLA。严格全史方案需要采购
[巨潮商业 API](http://webapi.cninfo.com.cn/)并确认历史版本语义，或建立版本化 PDF
解析和人工抽样复核流程。

## 6. 市值与执行数据

B070/B076 的可复用部分仍不足以闭环：

- B070 价格覆盖 1,310/1,310 标的、2,467,137 行，时间为
  `2018-01-02..2026-06-18`。
- 其中有 16,779 个停牌行、500 个发生过停牌的标的、52 条右删失路径。
- `adj_close` 与 `close` 逐行相等，没有调整因子、分红、拆并股或总股本字段。
- B076 市值覆盖 1,310 个标的，但从 2018 年开始，且是由换手率反推的**流通市值**；
  全公司归母净利润应配总市值，二者不能混用。

因此主窗口前四年缺价格/市值，公司行动 ledger 也不完整。用当前总股本乘历史价格会
产生 share-count lookahead。

## 7. 可复现性

本轮使用的 B068 基本面、B070 宇宙和价格、公告缓存、B076 市值以及 B082 指数快照
均被 `.gitignore` 的 `data/*` 规则排除。JSON 保存了每个本地输入的 SHA256，但 fresh
clone 没有这些文件，也没有一条具备版本固定与完整性校验的重建命令。

中证样本日期接口在独立复核中曾短暂返回 HTTP 200 空响应；runner 现对所有官方读取
执行最多三次的内容校验重试。恢复后两次正式输出逐字节一致，但在线端点稳定性仍应
作为 soft watch 保留。

所以即使其他数据门通过，当前也不能签为可复现研究。

## 8. 下一步数据采购规格

在以下四项全部到位前，继续禁止计算候选收益：

1. 中证授权的 H30269 自 2013 年起逐次生效成分、月末/调样权重、公司行动和除数。
2. 带首次公告时间和历史版本的归母净利润；累计值可严格还原单季，未来修订不会覆盖。
3. 2013 年起 PIT 总股本、总市值、中证一级行业、复权因子，覆盖退出和退市标的。
4. 原始可执行 OHLCV、停牌、涨跌停、现金分红、公司行动和退市处理；每期成员与 E/P
   覆盖至少 95%，并提供 fresh-clone 重建及哈希校验。

数据到位后先复制 H30269 股票篮子，并用 H20269 验证跟踪误差；复制失败时仍不得进入
E/P 候选回测。
