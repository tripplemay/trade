## 3. R1 阶跃性：144 月中 **0** 次违规

`ttm_step_without_filing = 0`（smoke 跑 3 个形成日；全量跑结果见
`docs/audits/B110-F002-panel.json`）。

判据：`TTM(i,t) ≠ TTM(i,t−1M)` 时，必存在某分量满足
`t−1M < f_ann_date ≤ t`。违反即意味着 as-of 漏进了未来 vintage、锚点抖动或缓存串期。

★**这是本设计里唯一直接检验 PIT 性的检查。**
