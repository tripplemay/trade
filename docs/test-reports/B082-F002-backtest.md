**口径分层（探针 §3/§4 驱动）：主 = H20269 全收益指数（TR，21.5y，无成本）；可实施 = 512890 ETF（sina 未复权价，7.46y，含成本/手数/费）。**

- **规则（焊死，spec §0/§3 ①，禁止扫参）**：spread = 指数股息率% − 十年国债% → **≥2.5% 满配 100% / 1.5–2.5% 半配 50% / <1.5% 低配 25%**。阈值是 `trade/strategies/cn_dividend_lowvol/parameters.py` 的模块常量，代码无任何 grid/optimise 路径。`parameter_hash = 66b22aa199b1c8ca…2128bc`。
- **股息率反推**：TR−PR 滚动 252 交易日（探针 §3；additive `TR_return − PR_return`，percent）。深度 2007-01→2026-07（前 252 日热身后），min 0.69% / mean 4.51% / max 12.86%。
- **执行**：月末读 spread，**T+1** 成交（指数层=次日 index 值；ETF 层=次日 open）。235 个月末：低配 124 / 半配 32 / 满配 79（当前 spread 2.95% → 满配）。
- **成本口径（ETF 层）**：佣金 2.5bp + 滑点 5bp 双边，**无印花税**（ETF 免，`CnCostModel(stamp_duty_bps=0)`）；100 份/手取整；管理+托管费 **0.5%/yr** 作 NAV 日衰减扣减。现金收益率 = 0（保守：真实现金/短债腿会赚≈十年国债，只会**帮**防守规则，故 0 是对规则不利的诚实选择）。

**读法（防守腿口径，非收益口径）：** 三档利差规则是一个**估值择时叠加层**——大涨压缩股息率利差后减仓，从而在随后崩盘（2008/2015）里已处低配。全周期它把最大回撤从 **−66.2% 削到 −40.5%（+25.7pp）**、Sharpe 微升（0.579→0.590），**代价是收益（CAGR 10.64%→7.49%，OOS 8.54%→6.34%）**。**规则对收益无增量，对尾部回撤有实质保护——这正是防守腿的设计目标。**

## 4. 诚实边界

1. **规则无收益增量**：三档利差规则在两口径均**不敌**买入持有的收益（主 −3.15pp CAGR，可实施 −3.6pp）。防守腿验收重点是回撤不是收益，**「规则无收益增量」是合法结论**，写进卡片（`oos_result=mixed`）。
2. **可实施层未复权失真**：512890 sina 价未复权 → CAGR 系统性低估 ~4.5%/yr（真实投资者拿到分红，见 §1 TR 口径 +10.6% 持有 CAGR）；ETF 层负 CAGR 是口径 artifact + 成本，**不是策略失效**。
3. **反推为指数口径股息率**：TR−PR 反推的是成分加权指数股息率，与 ETF 实际分红时点有差；利差信号分子分母同为指数/官方口径，自洽。
4. **2025 跑输**：策略/持有 +7.9% vs HS300 +21.2%——红利低波在 AI 牛中大幅跑输（spec 已知风险 + 拥挤度）；规则此期满配，跑输是**品种特性非规则 artifact**。
5. **窗口/方法边界**：主口径 21.5y 覆盖 2008/2015/2018 三轮熊 + 2022-24 调整 + 2025 跑输（风格周期覆盖佳）；CPCV-lite 为交错子样本（非全 CPCV，一 fold −5.76%）；现金收益率取 0（对规则不利的保守设定）。
