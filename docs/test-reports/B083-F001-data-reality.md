## 主信号源：`stock_yjyg_em`（业绩预告，东方财富）

实测 `stock_yjyg_em(date="YYYYMMDD")`（按报告期取），字段：
`序号 / 股票代码 / 股票简称 / 预测指标 / 业绩变动 / 预测数值 / 业绩变动幅度 / 业绩变动原因 / 预告类型 / 上年同期值 / **公告日期**`

**★命门通过 — 公告日期 = PIT 发布日**（非报告期末）：

| 报告期 | 预告数 | 公告日期范围（真发布日） |
|---|---|---|
| 2019-09-30 | 2304 | 2019-07-16 .. 2019-11-11 |
| 2021-09-30 | 2965 | 2021-07-09 .. 2022-03-14 |
| 2023-09-30 | 1013 | 2023-07-27 .. 2024-01-22 |
| 2024-09-30 | 598 | 2024-08-02 .. 2024-11-14 |

- **PIT feasible**：公告日期是真实**发布日**（跨报告期末前后散布，如 2024Q3 从 08-02 预披到 11-14）。
  PEAD 进场 = 公告日期 **T+1 open**，无前视。
- **盈余惊喜可算**：`预测数值`（预告净利润/中点）vs `上年同期值`（去年同期基准）→ 惊喜 = (预测−上年)/|上年|；
  `业绩变动幅度`(%) 已提供作交叉校验；`预告类型`（略增/预增/扭亏/首亏/预减/略减/续盈…）作分档。
- **深度**：≥2019（5+ 年，>MinTRL 量级）。
- **覆盖**：每报告期 598–2965 事件（近年 Q3 预告数下降 = 强制披露口径变化，非数据缺失；年报/一季报预告数更高）。

## 次信号源（secondary，F002 视需要）
- `stock_yjkb_em`（业绩快报）：正式快报 EPS/营收，比预告晚但更精确。
- `stock_yysj_em`（预约披露时间）：预约披露日 → 事件窗口预期（drift 前定位）。
- `stock_profit_forecast_em`（分析师一致预期）：SUE 分母的 secondary（覆盖够则用；先验用去年同期）。

## 诚实边界 / 风险
- **近年 Q3 预告数下降**（2965→598）：强制披露规则收紧，样本量按报告期波动 → F002 IC 分报告期报覆盖，别混池掩盖稀疏期。
- **涨跌停可执行性**：大惊喜（预增/扭亏）次日常一字涨停买不进 → F002 必做涨跌停分层 IC（B081 F003 `_limit_hit_names`）。
- **公告日 vs 数据日**：极少数预告可能盘中发布 → 保守用公告日 T+1 open（本报告口径）。

## 裁定：**GO**
