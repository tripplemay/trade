### 阻塞项

- 负向 API locale 验收失败：

```bash
curl --noproxy '*' -sS -X PUT http://127.0.0.1:8723/api/execution/account \
  -H "Cookie: authjs.session-token=<minted>" \
  -H 'Accept-Language: zh-CN' \
  -H 'Content-Type: application/json' \
  --data '{"cash":-1,"base_currency":"USD","positions":[]}'
```

返回：

```json
{"detail":[{"type":"greater_than_equal","loc":["body","cash"],"msg":"Input should be greater than or equal to 0","input":-1,"ctx":{"ge":0.0}}]}
```

- 同样请求 `PUT /api/execution/account?locale=en`，返回仍是相同英文 422。
- 这说明 F004/F006 当前只覆盖了 `HTTPException` 站点的 i18n，没有覆盖 Pydantic `RequestValidationError`。

## Spec Gap

- 当前实现符合第 205 行，不符合第 188 行和 `progress.json.generator_handoff` 的 Codex 验收清单。
