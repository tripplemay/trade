## 诚实 caveat（焊死）

- **first-look ≠ 可交易 edge**：IC/分组 spread 只看苗头，非全回测、非 verdict、不承诺赚钱（搭策略 + 去偏全回测是后续批次，若追）。
- **选择偏差（异动条件）**：龙虎榜事件条件于"已发生大波动/换手触发上榜" → 测的是**条件信号**，非"机构悄悄建仓"的无条件信号；混入动量/反转效应。
- **非单调（驼峰）**：分组均值非单调——中性分位远期收益最高、极端买卖两端均值回归；正 top−bottom 不代表单调"买得越多收益越高"。这是低 rank-IC（~0.02）的根因，反向加固 INCONCLUSIVE/偏否。
- **覆盖缺口（核心）**：19.2% 去偏覆盖偏大市值名；小盘主体未覆盖 = 残余偏差，结论**仅对较大名子集成立**。
- **无前视**：entry = 上榜次一交易日（LHB 收盘后披露，当日收盘买入=前视，已规避）。
- **去偏 integrity**：远期收益用 B070 含退市价格（幸存者-free），退市名负收益计入。
- **纯研究边界**：research-only / no-broker / no 真金 / no 自动下单 / 只读公开披露 / 无生产改动。
