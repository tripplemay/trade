## 3. 三个新功能详解

### 3.1 B054 — 界面完整中文化（交易 gate）

- **做什么**：把界面所有英文变中文，让你准确读懂交易建议。
- **根因**：早期 i18n 只翻了固定文字；动态/AI/后端生成的文字（推荐理由、回测/风险解释、策略说明、订单清单、报告正文）大量是英文。
- **三家族**：① LLM 生成文字（你主诉「标的下面的介绍」）② 后端硬编码英文串 30+（含订单清单）③ 前端漏译 + 报告正文。
- **你的决策**：LLM 文字**只生成中文**；范围**全部纳入**（含报告正文）。
- **状态**：Generator 已实现 F001-F004，Codex 验收发现还有英文残留（CAGR/SHARPE/报告标题等），正在 fixing 收尾。**它一 done + 部署 = 你的交易 gate 解除。**

### 3.2 B056 — 模拟盘 / Paper Trading（基建）

- **做什么**：一个**虚拟账户**，给虚拟本金，**忠实跟策略配置**，调仓日自动虚拟调仓，每日用真实价格 mark-to-market，显示**账户盈亏 + 每个资产盈亏**。
- **为什么是好东西**：它是「回测」和「实盘」之间的**前向桥**——

  | | 回答什么 | 时间 | 真金 |
  |---|---|---|---|
  | 回测 | 过去会怎样（可能过拟合）| 历史 | 无 |
  | **模拟盘** | 现在往前正在怎样（真·样本外）| 前向累积 | 无 |
  | 实盘 | 我的钱跟着策略 | 前向 | 有 |

  让你下真单前/中，看策略真刀真枪前向的表现——**backtest 过拟合的最佳解药**。
- **你的决策**：与实盘**并行**（不是 gate）；排 **B054 后 B055 前**；**真·基建**（一套引擎参数化，Master 先，B055/未来策略各插一个）；成本**真实计**（含费/滑点）；**前向**（从激活日起累积，不 seed 历史）。
- **测策略非测执行**：模拟盘按收盘价+成本测**策略本身**；你的真实执行质量由实盘 journal 衡量——两者分开。
- **UI（已定案，6 区块）**：① 顶部摘要（净值/总收益/今日/**vs SPY**）② 净值曲线 + SPY 叠加 ③ 每资产盈亏表 ④ 配置 vs 目标漂移 ⑤ 调仓记录（**简版**：日期+成本）⑥ 设置。复用现有组件，界面一致。
- **状态**：spec + UI 全定案，B054 一完就开工。

### 3.3 B055 — 进攻型「单市场个股选股」新模式（Phase 4 策略）

- **做什么**：一个**独立的进攻模式**（不进 Master 综合，Master=稳健核心 / 本模式=进攻卫星，并行）——自己选**个股**、单市场、月度调仓、追更高收益。
- **你的决策**：风格=**动量+质量**（质量过滤 + 动量排名 top 15）；市场=**美股先行**（财报数据现实，扩充时每市场一份不交叉）；频率=**月度**；集中=**top 15**；资金=**独立账户**（自己的现金/持仓/推荐/回测，与 Master 分开）；落地=**分期**。
- **分期**：
  - **P1（本批）**：美股引擎 + **独立回测**——你能先在回测页**看疗效再决定上实盘**。
  - P2（未来）：独立账户的推荐 + 执行闭环。
  - P3（未来）：多市场扩展（非美市场用纯价格动量）。
- **状态**：P1 spec 已备，排在 B056 后。

## 4. 贯穿的诚实原则（这套系统的底线，也是我给你把关的标准）

1. **多元化 > 更多 alpha**：加策略的价值在补不同的风险来源，不在堆更多同质 equity beta。
2. **频率 ≠ 收益**：高频只带成本；收益来自 个股 + 集中 + 进攻因子（同步带更高风险/更深回撤）。
3. **前向验证打假**：回测可能过拟合 → 模拟盘前向 + walk-forward 样本外，才知道策略真行不行。
4. **永久硬边界**：research-only / no-broker（你自己执行）/ no 收益预测 / 不可能状态 fail-fast（不静默改账目）。
